# gcd-cli

> Get the greatest common devisor of two numbers CLI

## Installation

```
$ npm install --global @knutkirkhorn/gcd-cli 
```

## Usage

```
$ gcd --help

        Usage
          $ gcd <number1> <number2>

        Examples
          $ gcd 12 20
          $ gcd 13 234 
```
